android-override-network-security-config
========================================

Rebuild APK to trust user CA certificates

Prerequisites
-------------

* [JDK](https://jdk.java.net/)
* [Apktool](https://ibotpeaches.github.io/Apktool/)

Usage
-----

```shell
$ aonsc <in.apk> <out.apk> [file.keystore]
```
